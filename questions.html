What is Python? ▼

Python is a programming language commonly used for buisiness intelligence, data science and artificial intelligence tasks.

What is Pandas? ▼

Pandas is a Python library for data aggregation.

What is Matplotlib? ▼

Matplotlib is a Python library for data visualization.

Can Python be used for aritificial intelligence tasks? ▼

Yes, Python can be used for artificial intelligence tasks. In particular, there are libraries such as pytorch or tensorflow for neural networks.

What is a Group By? ▼

The group by clause is a combination of split-apply-combine. First the data are split into groups. Then for each group an aggregation function is applied. Finally the aggregated groups are combined.
