### Pandas DataFrame - 2 dimensional data structure - It consists of - Columns - Series (potentially different data types) - Index - Think of it as: - Excel Spreadsheet (on steroids) - SQL Table - Dictionary of Series objects
### Creating a DataFrame - From dictionary ``` import pandas as pd data_dict = {"Country":["Afghanistan", "Angola"], "Region": ["Asia", "Africa"] , "GDP": [1969, 6029], "Age": [62, 64]} df = pd.DataFrame(data_dict) ``` - From Excel Spreadsheet - From CSV File - From database - From API - ...
Basic Operations
Inspect DataFrame ``` df.head(10) # First 10 elements ``` ``` df.tail(10) # Last 10 elements ``` ``` df.shape # Shape of the DataFrame ``` ``` df.dtypes # Data types of columns (Series) ``` Select single column (Series) ``` df['Region'] ``` Select columns (DataFrame) ``` df[['Region', 'GDP']] ```
Unique values of Series ``` df['Region'].unique() ``` Drop columns ``` df.drop(columns=['ratio']) ``` Rename columns ``` df.rename(columns={"GDP": "gdp", "Age": "life_expectancy"}) ```
Sort DataFrame ``` df.sort_values(by='GDP', ascending=True) ``` Filter by value ``` df[df['Region'] == 'Asia'] ``` Filter by multiple values ``` df[(df['Region'] == 'Asia') & (df['Age'] >= 80)] ```
Series arithmetics ``` df['GDP'] * 100 # multiply each element with 100 ``` ``` df['GDP'].min() df['GDP'].max() ``` ``` df['GDP'].mean() df['GDP'].median() df['GDP'].quantile(0.5) ``` Combine two Series ``` df['GDP'] / df['Age'] # combine 2 columns ``` Assign new column ``` # Create column from other columns df['ratio'] = df['GDP'] / df['Age'] # Create constant column df['constant'] = 3.14 ```
