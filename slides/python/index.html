Lists: a collection of items 
      
          # 3 item list of numbers
          my_list = [1, 2, 3]

Acces by index (Start at 0) 
      
            my_list[0] # 1 
            my_list[1] # 2

List operations 
      
            min(my_list) # 1 
            max(my_list) # 3
            sum(my_list) # 6
            len(my_list) # 3
Iterate through list with for loop 
      
          for element in my_list:
            # need to indent!
            print(element)
Manipulate lists 
      
          result = [] 
          for x in my_list:
            result.append(x+1)
Syntactic sugar: list comprehensions 
      
          result = [x+1 for x in my_list]

Hello world as function

          print_something("Hello World")
Add two numbers 
               
            def add_numbers(x, y):
                return x + y
            add_numbers(3, 4) # 7
Defaul values 
               
            def add_numbers(x, y=1):
                return x + y
            add_numbers(3) # 4 same as add_numbers(3,1)
            add_numbers(3,4) # 7
